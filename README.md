# golf-world-cup-challenge

Static scoring page for the World Cup Challenge — the Newton Comm Monday Men's
League's group stage contest. Players compete hole-by-hole against net par
across 9 holes; ties are broken by a Penalty Shootout, then Sudden Death.

Deployed via Cloudflare Workers static assets, auto-building on every push
to `main`. See `World_Cup_Challenge.html` for the app and
`World_Cup_Challenge_Rules.md` for the full rules.
